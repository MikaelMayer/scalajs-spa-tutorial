Scala.js SPA-tutorial

Debugging

Scala.js supports source maps, making debugging your code in the browser quite easy. You can set breakpoints in the original source-code and inspect local variables etc. just like in a real IDE debugger. See the documentation for your browser developer tools for more information.

Setting source maps

We will need a few configuration settings in our build file to make sure source maps are correctly found.

sourceMapsBase := baseDirectory.value / ".." in the shared project sourceMapsDirectories += sharedJS.base / ".." in the client project

Actual debugging
